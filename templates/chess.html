Twitter Plays Chess

What is Twitter Plays Chess?

Ever heard of Twitch Plays Pokemon? A game of Pokemon Red was live streamed to the Twitch platform, where viewers would input commands into the stream's chat room, which would get executed by the background program and would then be displayed on the Twitch stream.
Twitter Plays Chess works the same way, except users are inputting commands to the program via tweets on Twitter instead of messages on Twitch.

To simplify this, the program takes input from a tweet and uses that to make a chess move.

Prerequisites:

Because Twitter Plays Chess involves using Twitter to send moves, there are prerequisites for the user to be able to participate in the Chess game:
  1. Have access any type of computer device, such as desktop PC, laptop, tablet or smartphone
  2. Have access to internet which does not block access to Twitter

How to participate:

Users always play as the white side.
This program takes chess moves in UCI format. This means that instead of typing traditional chess notation such as "Nf3" to make this move:
The input should be something like "g1f3", which means to move the piece currently in the square g1 to the square f3. This can be broken into two parts:
  1. The previous square of the piece, in this case "g1"
  2. The square the user wants the piece to move to, in this case "f3"
Tweets must be in the format of:

@bwbchess #bwbchess (last move the computer made) (user's move)

where the @ mention, hashtag and both moves are separated by a space.
When playing the very first move in the game, the user's input must be:

@bwbchess #bwbchess none (user's move)

If it is not exactly like this, simply put, nothing will happen on the chess board.
